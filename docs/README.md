# ONE DAY - 输赢之间

## 游戏简介

一款文字向的互动网页游戏，通过简单的故事选择和隐藏的文字编辑机制，探讨"输"与"赢"的相对性，启发玩家思考生活中成败的真正意义。

## 核心特性

- **可编辑的文字系统**：游戏中的"输"和"赢"字可以修改，这是通关的关键
- **多重结局**：两种不同的通关方式，带来不同层次的思考
- **渐进式背景解锁**：
  - 初始背景：蓝底水彩哭脸
  - 第二背景：水彩笑脸（1种通关方式解锁）
  - 第三背景：哭笑拼接脸（2种通关方式解锁）
- **成就系统**：追踪通关进度，展示解锁状态

## 游戏流程

1. **首页**：查看成就，了解游戏目标
2. **目标页**：阅读目标（发现可以修改"赢"字）
3. **故事页1-6**：做出选择，体验一天的经历
4. **结局页**：发现可以修改"输"字
5. **通关**：返回主页，解锁新背景

## 通关条件

### 方式一
在目标页将"你要赢"改为"你要输"，完成所有故事选择后返回主页

### 方式二
正常游玩到结局页，将结局中的"你输了"改为"你赢了"后返回主页

## 技术栈

- **前端框架**：Vue.js 3
- **构建工具**：Vite
- **状态管理**：Pinia
- **路由**：Vue Router
- **样式方案**：CSS3
- **状态存储**：Local Storage（保存通关进度）
- **部署方式**：静态网页

## 游戏特色

- 独特的文字编辑机制，没有明显提示，需要玩家细心观察
- 哲学性的故事内容，引发对生活态度的思考
- 简洁的视觉风格，突出文字内容
- 低门槛，高深度的游戏体验

## 游戏说明

本游戏是文字向游戏，请仔细阅读文本，完成游戏目标，通关游戏吧。希望游戏能让您有所启发，有所收获。

**游玩建议**：建议一个人时游玩，游玩时仔细阅读，认真思考。
